# ATM Banking System

## Project Overview

ATM Banking System is a console-based banking application developed using Python and MySQL. The project simulates basic ATM operations and stores account information in a MySQL database.

## Technologies Used

* Python
* MySQL
* PyCharm
* Git
* GitHub

## Features

* Create Account
* User Login using PIN
* Deposit Money
* Withdraw Money
* Balance Enquiry
* Change PIN
* Exit System

## Database

Database Name:

```text
atm_system
```

Table:

```text
accounts
```

Fields:

```text
acc_no
name
pin
balance
```
